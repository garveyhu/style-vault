Category Nav · 六款方向对比

给分类导航换一个更有质感的样式

每款都把「模块」设为 active，hover 会有微动效。选出一款告诉我，比如 "V3"。

V0 · 当前
风格 页面 模块 组件 原语
当前样式——大字 + 渐变下划线。你觉得"太普通"，所以有了下面这六款。
V1 · Pill Group
风格 页面 模块 组件 原语
灰底容器 + 黑底 active pill，干净稳健。常见于 Linear / Arc 等新派 SaaS。整体紧凑、有边界，不漂浮。
V2 · 衬线 + 数字
01风格 02页面 03模块 04组件 05原语
Fraunces 衬线 22px + Plex Mono 小号编号，active 变深 + cyan 数字。最像"杂志目录"，编辑/文艺气质最强。
V3 · pageflows 同款深色
风格 页面 模块 组件 原语
深色 segmented + 白 active。浅色站上深色 pill = 视觉重量中心，权威感。
V4 · 卡片 tabs（编号 + 计数）
01 风格 1 02 页面 1 03 模块 2 04 组件 1 05 原语 2
每个 tab 是一张有边框的小卡片，内嵌编号 + label + 计数。active 整卡黑色 + cyan 编号，上浮 1px 带阴影。信息量最大。
V5 · 左侧竖线 accent
风格 页面 模块 组件 原语
active 的 tab 左侧长出一根 cyan→slate 渐变的小竖线。极简、气质安静，像文档侧边栏。
V6 · 超大衬线 + 上标数字
01风格 02页面 03模块 04组件 05原语
Fraunces 28px + 上标数字。inactive 极淡 slate-300，active slate-900 + 加粗。像品牌主视觉，存在感最强。

告诉我比如 V2 或 V5，我就落地到代码里。
