选择提示

哪个适合你？

IF 产品 < 6 条
V3 / V4

产品数不多，要做出"有分量"的感觉。V3 用 Hero 突出主推；V4 全部横向长卡，阅读感强。

IF 产品 6–20 条
V2 / V6

中等规模最适合 Bento（V2）打破节奏，或封面墙（V6）让设计本身说话。

IF 产品 > 20 条
V5

数量上来之后必须按 category 分组，用户才扫得动。V5 天然有"目录感"。
